# Travel Buddy

Multi-screen React Native app dengan React Navigation.

## Features
- Bottom Tab Navigation (Beranda, Cari, Favorit)
- Stack Navigator di setiap tab (HomeScreen → DetailScreen)
- Route params untuk passing data destinasi
- FlatList + Horizontal Scroll destinations
- Search filter by nama & lokasi
- Add to Favorites + tab badge
- @expo/vector-icons (Ionicons) untuk tab & UI icons

## Tech Stack
- React Native + Expo
- React Navigation 6
- React Context API
- StyleSheet
- @expo/vector-icons

## How to Run
1. npm install
2. npx expo start
3. Scan QR code di Expo Go
